## 📝 Notas para Producción

### Estructura recomendada:
1. **Hook** (0-30s): Promesa + credibilidad
2. **Problema** (30s-2m): Por qué importa
3. **Contenido** (2m-12m): El "cómo"
4. **Prueba** (12-15m): Resultados/casos
5. **CTA** (15m+): Suscribirse + próximo video

- [ ] Outline escrito
- [ ] Thumbnail idea
- [ ] 3 hooks alternativos
