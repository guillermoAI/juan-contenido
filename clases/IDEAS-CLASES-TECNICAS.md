## 📋 SOPs (Procedimientos)

### Contenido Diario
```
SOP: Publicar un Reel
1. Revisar calendario de contenido
2. Grabar (máx 3 tomas)
3. Editar en CapCut (15-20 min máx)
4. Escribir caption con hook
5. Elegir cover
6. Publicar + 3 hashtags
7. Compartir a stories
Tiempo total: 45 min
```

### Gestión de DMs
```
SOP: Responder DMs
1. Revisar DMs 3x/día (9am, 2pm, 7pm)
2. Priorizar: Leads calientes > Preguntas > Resto
3. Responder en <4 horas
4. Si es lead: calificar con preguntas
5. Si califica: agendar call
6. Trackear en CRM/Notion
```

### Onboarding Cliente
```
SOP: Nuevo Cliente
1. Recibir pago confirmado
2. Enviar email de bienvenida (template)
3. Crear carpeta en Drive
4. Agendar call de kick-off (48h máx)
5. Enviar formulario de intake
6. Preparar diagnóstico inicial
7. Call de kick-off
```

### Preparar Call de Venta
```
SOP: Antes de Call
1. Revisar perfil de IG del lead
2. Ver sus últimos posts/stories
3. Revisar notas del setter (si hay)
4. Preparar 2-3 preguntas personalizadas
5. Tener pricing listo
6. Tener calendario abierto para onboarding
```
