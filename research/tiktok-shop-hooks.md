## Hooks que Funcionan en TikTok Shop (Patrones)

### Hook Pattern 1: Números Específicos
- "Vendí 1,100 garrafas en 3 horas"
- "Gané €X este mes con TikTok Shop"
- "Esta cosa de €15 me ahorra €200 al mes"

### Hook Pattern 2: Contrarian
- "TikTok Shop NO es una red social"
- "No necesitas ser influencer para vender"
- "Todo el mundo está haciendo esto MAL"

### Hook Pattern 3: Proceso Real
- "Así es como empecé desde CERO"
- "Mi primera venta fue un desastre, pero..."
- "Nadie te cuenta esto sobre TikTok Shop"

### Hook Pattern 4: Prueba Social Visual
- Mostrar dashboard de ventas
- Mostrar notificaciones de pedidos
- Mostrar productos físicos vendidos

## Errores Comunes (Por qué Alex no tiene tracción)

1. **Contenido genérico** - no diferencia de otros 1000 creadores
2. **No muestra resultados reales** - tiene €20k/mes pero no lo usa
3. **Falta de urgencia/escasez** - no hay razón para actuar AHORA
4. **Hooks débiles** - primeros 2 segundos no capturan
5. **No documenta proceso** - la gente quiere ver el "behind the scenes"
