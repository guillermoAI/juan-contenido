High-ticket = €2,000+ por cliente

## Embudo 1: Profile Funnel (Nik Setting)

**El modelo que usas actualmente.**

```
Contenido Viral (Reels/YT)
         ↓
    Perfil Optimizado
         ↓
    DM / CTA en Bio
         ↓
    Conversación
         ↓
    Call de Venta
         ↓
    Cliente €3-5k
```

### Métricas Típicas
| Paso | Conversión |
|------|------------|
| Views → Perfil | 1-3% |
| Perfil → DM | 5-15% |
| DM → Call | 20-40% |
| Call → Cliente | 20-40% |

### Claves del Éxito
1. **Contenido polarizante** (no tibio)
2. **Bio clara** con promesa específica
3. **Highlights** que precalifican
4. **Respuesta rápida** a DMs (<1h)
5. **Proceso de calificación** antes de call

### Ventajas
- Sin ads, sin tech compleja
- Funciona con pocos seguidores
- Relación directa

### Desventajas
- No escala infinitamente
- Requiere tu tiempo en DMs
- Dependes del algoritmo

## Embudo 2: VSL Funnel

```
Ad / Contenido Orgánico
         ↓
    Landing Page
         ↓
    VSL (15-45 min)
         ↓
    Aplicación
         ↓
    Call de Venta
         ↓
    Cliente
```

### Componentes
1. **VSL (Video Sales Letter)**
   - Hook (0-30s): Problema + promesa
   - Historia (30s-5m): Tu journey
   - Contenido (5-20m): Framework/valor
   - Oferta (20-30m): Qué incluye
   - CTA (30m+): Aplicar ahora

2. **Página de Aplicación**
   - Typeform/Tally con preguntas de calificación
   - Facturación actual, nicho, urgencia
   - Booking automático si califica

### Métricas Típicas
| Paso | Conversión |
|------|------------|
| Click → Landing | - |
| Landing → VSL 50% | 30-50% |
| VSL 50% → Aplicación | 5-15% |
| Aplicación → Call | 50-70% |
| Call → Cliente | 20-40% |

### Claves
- VSL debe ser BUENO (no aburrir)
- Aplicación filtra tire-kickers
- Setter puede precalificar antes de call

## Embudo 3: Webinar Funnel

```
Ad / Contenido
         ↓
    Registro Webinar
         ↓
    Webinar (60-90 min)
         ↓
    Oferta al Final
         ↓
    Call o Compra Directa
```

### Estructura del Webinar
1. **Intro (5m):** Hook, credibilidad rápida
2. **Contenido (40-50m):** 3-4 puntos de valor real
3. **Transición (5m):** "Ahora, para los que quieren ir más rápido..."
4. **Oferta (15-20m):** Stack completo
5. **Q&A (15m):** Manejar objeciones en vivo

### Tipos
- **Live:** Más conversión, menos escalable
- **Evergreen:** Automatizado, menos conversión
- **Híbrido:** Live grabado que se reutiliza

### Métricas
| Paso | Conversión |
|------|------------|
| Landing → Registro | 20-40% |
| Registro → Asiste | 20-40% |
| Asiste → Oferta | 5-15% |

## Embudo 4: Lead Magnet + Nurture

```
Ad / Contenido
         ↓
    Lead Magnet (PDF, Mini-Curso)
         ↓
    Email Sequence (5-14 días)
         ↓
    Call/Oferta
         ↓
    Cliente
```

### Lead Magnets que Funcionan
- Checklist específica ("Los 7 errores de...")
- Mini-curso (3-5 videos cortos)
- Template/Script
- Caso de estudio detallado
- Calculadora/Assessment

### Secuencia de Emails
```
Día 0: Entrega + Quick Win
Día 1: Tu historia
Día 2: Caso de estudio
Día 3: Error común #1
Día 4: Error común #2
Día 5: El sistema completo
Día 6: Oferta soft
Día 7: Oferta directa + urgencia
```

## Embudo 5: Appointment Setting (Escala)

```
Contenido Masivo
         ↓
    Leads Entrantes (DMs, Comments)
         ↓
    Setter (califica + agenda)
         ↓
    Closer (vende)
         ↓
    Cliente
```

### Roles
- **Setter:** Responde DMs, califica, agenda calls
- **Closer:** Solo cierra ventas

### Compensación Típica
- Setter: €1,000-2,000/mes base + bonus por calls agendadas
- Closer: 10-20% comisión

### Cuándo Implementar
- Cuando tienes >50 leads/semana
- Cuando tu tiempo en ventas > tiempo en delivery
- Facturación >€20k/mes consistente

## ¿Cuál Usar?

### Estás empezando (<€10k/mes)
→ **Profile Funnel**
- Sin inversión en ads
- Aprendes vendiendo tú
- Construyes audiencia

### Ya tienes tracción (€10-30k/mes)
→ **Profile + VSL**
- Escala orgánico
- Añade tráfico pagado
- Automatiza calificación

### Escalando (>€30k/mes)
→ **Full Funnel + Team**
- Setters + Closers
- Múltiples fuentes de tráfico
- Webinars periódicos
