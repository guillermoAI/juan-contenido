# 🎯 Embudos de Venta (Funnels)

Guía completa de los principales embudos usados en consultoría y info-productos.

## Tipos de Embudos

| Embudo | Complejidad | Ticket | Mejor para |
|--------|-------------|--------|------------|
| [Profile Funnel](profile-funnel.md) | Baja | €2k-10k | Consultores individuales |
| [Skool Funnel](skool-funnel.md) | Media | €50-500/mes | Comunidades + cursos |
| [VSL Funnel](vsl-funnel.md) | Alta | €500-5k | Info-productos |
| [Webinar Funnel](webinar-funnel.md) | Alta | €1k-10k+ | High-ticket |

## ¿Cuál elegir?

### Si estás empezando (0-10 clientes):
→ **Profile Funnel** - Mínima fricción, resultados rápidos

### Si tienes audiencia (1k+ seguidores):
→ **Skool** para recurrencia o **VSL** para productos

### Si vendes high-ticket (€5k+):
→ **Webinar** o **Profile + Call**
